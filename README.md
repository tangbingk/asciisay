## Install
```bash
git clone https://github.com/tangbingk/asciisay.git
cd asciisay
sudo npm install . -g
```

## Run
```bash
asciisay -h

	Usage: asciisay [options]

	Options:
	-V, --version       output the version number
	-i, --index <n>     ascii art index, default is random (default: -1)
	-t, --type <value>  [quotes|jokes|tang|song] (default: "tang")
	-h, --help          output usage information
			https://github.com/tangbingk/asciisay.git
```
